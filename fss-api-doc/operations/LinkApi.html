Create link

Creates a link to a file or folder in HCP Anywhere.

URI

/fss/public/link/create

HTTP Method

POST

Parameters

  • Request Body:
    Request body for create link▾

    Request body parameters needed to create a link. This request body needs to be JSON.

Example Request

Here is an example of a create link request in cURL. In this example, the link is created to a folder called "Team-Projects" and and the access token is "abc=". In an actual example, the access token is much longer. 

 curl -kiX POST https://anywhere.example.com/fss/public/link/create -H "Accept: application/json" -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "X-HCPAW-FSS-API-VERSION: 2.1.1" -d '{"path": "/Team-Projects", "expirationDays": 30, "public": true, "accessCode": true, "permissions": ['READ']}'

Description of Response

Response body for create link▾

Response Codes

  • 200 - Operation successful.
  • 400 - Bad request. The error message contained in the reponse body includes more information.
  • 401 - Bad or expired access token.
  • 403 - Access to the resource was forbidden while trying to create the link.
  • 500 - Internal server error.
  • 503 - The server was temporarily unable to create the link. The request should be retried.
