• url (required) : String
    The url of the link.
  • path (required) : String
    The path to the entry for which the link was created.
  • public (required) : Boolean
    Whether the link is publicly accessible (accessible without first entering login credentials).
  • accessCode (optional) : String
    The access code for the link.
  • expirationDate (required) : Long
    The number of milliseconds (since epoch) until the link expires.
  • permission (required) : String
    The type of operations that may be performed on the file or folder when accessed through the link.
    Enum:
    READ - The user can view and download files.
    UPLOAD - If the link is to a folder, the user can upload files to the folder.
