• entries (required) : Object
    The contents (files and folders) of the folder that is being searched. The number of entries returned can be at most the number of entries specified in the maxResults parameter.
  • type (optional) : String
    The type of entry (file or folder).
  • parent (optional) : String
    The path to the entry.
  • name (optional) : String
    The name of the entry.
  • changeTime (optional) : Long
    The time (in milliseconds) the entry was last modified since epoch.
  • size (optional) : Long
    If the entry is a file, the size of the file in bytes.
  • hash (optional) : String
    If the entry is a file, the hex encoded sha-384 hash of the file.
  • access (optional) : String
    The type of operations a user can perform on the entry.
    Enum:
    NO ACCESS - The user cannot access the file or folder.
    VIEWER - The user can view and download the file or folder. However, if the entry is a folder, the user cannot upload files to the folder.
    COLLABORATOR - The user can download, upload, move, and delete files and folders.
  • etag (optional) : String
    The etag of the entry. The etag identifies the version of the file or folder and is supplied as a parameter in some of the other File Sync and Share API endpoints.
