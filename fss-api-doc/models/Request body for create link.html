• path (required) : String
    The path to the file or folder for which to create a link.
  • expirationDays (required) : Integer
    The number of days until the link expires. The settings on the Management Console determine the default number of days until the link expires.
  • publicLink (optional) : Boolean
    Whether the link is publicly accessible (accessible without first providing login credentials). The settings on the Management Console determine the default accessibility of a link (public or internal).
  • accessCode (optional) : Boolean
    Whether the server should create an access code for the link. If a link has an access code, the file or folder cannot be accessed without first providing the access code to HCP Anywhere.
  • permissions (required) : String
    The access permissions for the file or folder that the link points to.
    Enum:
    READ - The user can view and download files.
    UPLOAD - If the link is to a folder, the user can upload files to the folder.
