Return Body for getUserInfo

  • username (required) : String
    The username of the user.
  • quotaBytes (required) : Long
    The total amount of space (in bytes) the user has for adding files to HCP Anywhere.
  • usageBytes (required) : Long
    The amount of space (in bytes) the user is currently using.
  • email (required) : String
    The email address of the user.
  • linkSettings (required) : LinkSettings
