• sourcePath (required) : String
    Current path of the file or folder to move. The last entry in the path should be the name of the file or folder to move.
  • destinationPath (required) : String
    The path where the file or folder will be moved. The last entry in the path should be the name of the file or folder to move. If a different name is entered for the last entry in the path, the file or folder is renamed.
  • etag (required) : String
    The etag of the file or folder. The etag identifies the version of the file or folder and is part of the file or folder metadata.
  • createParents (optional) : Boolean
    Whether the operation should create any missing folders in the given destinationPath. If false is entered for this parameter, then the operation fails if any folders in the given destinationPath do not exist.
