• path (required) : String

    The path of the folder to create. The last entry in the path should be the name of the folder to create.

    If a folder with the same name already exists at the given path, the existing folder is returned.

  • createParents (optional) : Boolean
    Whether the operation should create any missing folders in the given path. If false is entered for this parameter, then the operation fails if any folders in the given path do not exist.
