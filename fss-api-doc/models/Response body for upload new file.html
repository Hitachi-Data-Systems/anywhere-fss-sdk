• type (required) : String
    The type of entry (file or folder). For an upload file response, this will always be file.
  • parent (required) : String
    The path to the file.
  • name (required) : String
    The name of the file.
  • changeTime (required) : Long
    The time (in milliseconds) the file was last modified since epoch.
  • size (required) : Long
    The size of the file in bytes.
  • hash (required) : String
    The hex encoded sha-384 hash of the file.
  • access (required) : String
    The type of operations a user can perform on the entry.
    Enum:
    NO ACCESS - The user cannot access the file.
    VIEWER - The user can view and download the file.
    COLLABORATOR - The user can view, download, move, and delete file.
  • etag (required) : String
    The etag of the entry. The etag identifies the version of the file or folder and is supplied as a parameter in some of the other File Sync and Share API endpoints.
