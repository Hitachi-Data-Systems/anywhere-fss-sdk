Folder Info

  • type (required) : String
    The type of entry (file or folder). For a createFolder request, this will always be folder.
  • parent (required) : String
    The path to the folder.
  • name (required) : String
    The name of the folder.
  • changeTime (required) : Long
    The time (in milliseconds) the folder was last modified.
  • access (required) : String
    The type of operations the user can perform on the folder.
    Enum:
    NO ACCESS - The user cannot access the folder.
    VIEWER - The user can view and download the files and folders in the folder. However, the user cannot upload files to the folder.
    COLLABORATOR - The user can download, upload, move, and delete the files and folders in the folder.
  • etag (required) : String
    The etag of the entry. The etag identifies the version of the file or folder and is supplied as a parameter in some of the other File Sync and Share methods.
