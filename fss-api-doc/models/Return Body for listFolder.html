Return Body for listFolder

  • pageToken (required) : String
    The token used to get the next page of entries in the folder.
  • entries (required) : List
    The contents of the given folder. The number of entries returned can be up to the number of entries specified in the pageSize parameter.

    If there are more entries in the folder, a page token is also returned. The page token is used to get the next page of entries in the folder. By deafault, this is 100.
