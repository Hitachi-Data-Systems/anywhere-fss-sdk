Request Body for createLink

  • path (required) : String
    The path to the file or folder for which to create a link.
  • expirationDays (required) : Integer
    The number of days until the link expires. The default number of days is determined by the settings in the Management Console.
  • publicLink (optional) : Boolean
    Whether the link is publicly accessible (accessible without first providing login credentials). The default accessibility is determined by the settings in the Management Console.
  • accessCode (optional) : Boolean
    Whether to create an access code for the link. If a link has an access code, the file or folder cannot be accessed without first providing the access code.
  • permissions (required) : String
    The access permissions for the file or folder that the link points to.
    Enum:
    READ - The user can view and download files.
    Upload - If the link is to a folder, the user can upload files to the folder.
