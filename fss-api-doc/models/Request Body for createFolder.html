Request Body for createFolder

  • path (required) : String
    The path in HCP where to create the folder. The last entry in the path should be the name of the folder to create.

    If a folder with the same name already exisits at the given path, the existing folder is returned.

  • createParents (optional) : Boolean
    Whether the operation should create any missing folders in the given path. If false is entered for this parameter, then the operation fails if any folders in the given path do not exist.
