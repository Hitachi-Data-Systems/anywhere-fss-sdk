• path (required) : String
    The path of the file or folder to delete. The last entry in the path should be the name of the file or folder that is being deleted.
  • recursive (optional) : Boolean
    If the entry is a folder, whether to delete all the files and folders within the folder. If this is set to false, and there are files and folders in the folder that is being deleted, the operation fails.
  • etag (required) : String
    The etag of the file or folder to delete. The etag identifies the version of the file or folder and is part of the file or folder metadata.
