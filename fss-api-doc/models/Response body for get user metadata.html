• username (required) : String
    The username of the user.
  • quotaBytes (required) : Long
    The user's total storage quota (in bytes).
  • usageBytes (required) : Long
    The amount of quota (in bytes) the user is currently using.
  • email (required) : String
    The email address of the user.
  • linkSettings (required) : Object
    The user's default link settings.
  • sharingEnabled (required) : Boolean
    Whether link sharing is enabled on the server.
  • publicSharingEnabled (required) : Boolean
    Whether public link sharing is enabled on the server.
  • uploadEnabled (required) : Boolean
    Whether upload link sharing is enabled on the server.
  • publicUploadEnabled (required) : Boolean
    Whether public upload link sharing is enabled on the server.
  • sharingDefaultPublic (required) : Boolean
    Whether sharing links publicly is set to default on the server. If false, sharing links internally is the default.
  • defaultDaysToShare (required) : Integer
    The default number of days a link created by the user remains active.
  • maxDaysToShare (required) : Integer
    The maximum number of days a link created by the user can remain active.
