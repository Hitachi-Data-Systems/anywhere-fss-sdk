Request Body for listFolder

  • path (required) : String
    Path to the folder. The last entry in the path should be the folder to list.

    This parameter is only required on the initial request. If there are requests for subsquent results from the folder, the page token returned from the previous request should be placed in the request body.

  • pageToken (required) : String
    Token returned in the body of the previous response. A token is only returned if there is additional content (files and folders) in the folder that was not returned in the last response.

    This parameter should not be provided in the first request to list the contents of a folder. Once all the contents of the given folder have been listed, a page token is not returned.

  • pageSize (optional) : Integer
    The maximum number of entries to return.
