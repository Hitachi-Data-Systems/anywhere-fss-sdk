List restore points

This request lists the restore points available to a user. A restore point is a snapshot of the filesystem at a certain point in time. Restore points can be used in a number of other requests. For example, you can a restore point in a path/zip request to download files and folders as they were at a previous point in time.

The restore points available to a user are determined by the HCP Anywhere administrator. In the HCP Anywhere Management Console, restore points are set through extended versioning.

URI

/fss/public/user/restorePoints/list

HTTP Method

POST

Parameters

There are no parameters for a user/restorePoints/list request.

Example Request

This section contains an example of a user/restorePoints/list request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/user/restorePoints/list -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.0"

Example Response

This section contains an example of a response to a successful user/restorePoints/list request.

{
    "restorePoints": [
      {
          "point": 1481173199999
      },
      {
          "point": 1481086799999
      }
   ]
}

Description of Response

This section describes the contents of a user/restorePoints/list response.

Response content:

•resoterPoints (required): Object

A list of the user's available restore points.

opoint (required): Long

A restore point available to the user.
