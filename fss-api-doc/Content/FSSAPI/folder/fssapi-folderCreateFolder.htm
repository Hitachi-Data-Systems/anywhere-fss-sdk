Create folder

This request creates a folder.

URI

/fss/public/folder/create

HTTP Method

POST

Parameters

The folder/create endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•path (required) : String

The path of the folder you want to create.

•createParents (optional) : Boolean

If true, the request creates any folders that do not exist in the path you specified. If false, the request fails if any folders in the path do not exist. The default is false.

Example Request

This section contains an example of a folder/create request in cURL. In this example, the folder being created is "Sales-Reports", and the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/folder/create -H "Accept: application/json" -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.0" -d '{"path": "/Work-Projects/Sales-Reports", "createParents": true}'

Example Response

This section of a response for a successful folder/create request. In an actual response, the etag is much longer.

{
    "type": "FOLDER",
    "parent": "/Work-Projects",
    "name": "Sales-Reports",
    "changeTime": 1456243695357,
    "access": "COLLABORATOR",
    "etag": "kaf=="
}

Description of Response

This section describes the contents of a folder/create response.

Response content:

•type (required) : String

The type of item (FILE or FOLDER). For this response, the type is always FOLDER.

•parent (required) : String

The path to the folder that contains the folder.

•name (required) : String

The name of the folder.

•changeTime (required) : Long

The time at which the folder was last accessed or modified. This time is returned in Unix format.

•etag (required) : String

The etag of the folder. The etag identifies the version of the folder and is supplied as a parameter in some of the other File Sync and Share API endpoints.

•state (required) : Enum

The state of the folder (CREATE or DELETE).
