List members in AD group

This request lists the members of an Active Directory group.

Tip:  You can invite a groups to shared or team folders using a share/invite request.

URI

/fss/public/share/listGroupMembers

HTTP Method

POST

Parameters

The share/listGroupMembers endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•name (required): String

The name of the Active Directory group.

•providerId (optional): String

The unique identifying string assigned to the groups's authentication provider (AD or SAML) by HCP Anywhere. You can acquire the uniqueId of a group using a share/invite/search request.

Example Request

This section contains an example of a share/listGroupMembers request in cURL. In this request, the access token is "abc=". In an actual request, the access token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/listGroupMembers -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 4.0.0" -d '{"name": "Sales Team", "providerId": "2305843009213653953"}'

Example Response

This section contains an example of response to a successful share/listGroupMembers request.

{
    "groupMembers":[
       {
            {
        "name": "jsmith",
        "displayName":"Alex Bloom",
        "domain": "domain.example.com",
    {
        "name": "tlomas",
        "displayName":"Tyler Lomas",
        "domain": "domain.example.com",
      }
   ]
}

Description of Response

This section describes the contents of a share/listGroupMembers response.

Response content:

•groupMembers (required) : Object

A list of the members in the Active Directory group:

oname (required) : String

The user's username. The username is acquired from the identity provider (SAML or AD) of that user.

odisplayName (required) : String

The user's display name. The display name is acquired from the authentication provider (SAML or AD) of that user.

odomain (required): String

The Active Directory domain that contains the user's user account.
