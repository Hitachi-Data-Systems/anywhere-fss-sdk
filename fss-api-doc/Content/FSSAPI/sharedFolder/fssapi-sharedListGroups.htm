List groups in folder

This request lists the groups (current and former) in a shared or team folder.

URI

/fss/public/share/groups

HTTP Method

POST

Parameters

The share/groups endpoint requires only request body parameters. The request body needs to be in JSON format.

Request body parameters:

•path (required) : String

The path to the shared or team folder.

Example Request

This section contains an example of a share/groups request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/groups -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.3" -d '{"path": "/Sales Project"}'

Description of Response

This section describes the contents of a share/groups response.

Response content:

•members (required) : Object

A list of groups in the shared or team folder:

oname (required) : String

The name of the group. The name of the group is acquired from the identity provider (SAML or AD) of the HCP Anywhere system.

ouniqueId (required) : String

The unique identifying string assigned to the group by the group's authentication provider (AD or SAML). For AD, this is the user's Security Identifier (SID).

oproviderId (required) : String

The unique identifying string assigned to the group's authentication provider (AD or SAML) by HCP Anywhere.

orole (required) : Enum

The folder role of the group (VIEWER, COLLABORATOR, or MANAGER). The role identifies the operations a user can perform in the folder.

ostate (required) : String

The group's state in the folder (INVITED, MEMBER, FORMER_MEMBER).

odomain (required): String

The Active Directory domain that contains the group's user account.

oaccountType (required): Enum

Identifies whether the returned object is an individual person or a group (USER or GROUP).

ogroupInfo (required): Object

This object lists some additional information about the group:

–displayName : String

The group's display name. The display name is acquired from the authentication provider (SAML or AD) of the HCP Anywhere system.
