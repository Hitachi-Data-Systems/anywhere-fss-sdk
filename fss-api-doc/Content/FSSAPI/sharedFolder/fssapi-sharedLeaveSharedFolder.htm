Leave shared folder

This request leaves a shared or team folder

When you leave a shared or team folder, the folder is removed from your HCP Anywhere account.

If you're the last Manager of a team folder, you are prevented from leaving the folder. However, you can:

• Invite another user to be a Manager in the shared folder using a share/invite request.

•Delete the team folder for all the members using a share/team/delete request.

URI

/fss/public/share/leave

HTTP Method

POST

Parameters

The share/leave endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•path (required) : String

The path to the shared or team folder to leave.

Example Request

This section contains an example of a share/leave request in cURL. In this request, the access token is "abc=". In an actual request, the access token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/leave -H "Accept: application/json" -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "X-HCPAW-FSS-API-VERSION: 4.0.0" -d '{"path": "/Last Year's Projects"}'

Example Response

For a successful share/leave request, no information is returned in the response body.
