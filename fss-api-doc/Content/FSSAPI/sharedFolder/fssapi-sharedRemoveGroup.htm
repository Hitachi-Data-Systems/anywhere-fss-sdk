Remove group

This request removes a group from a shared or team folder.

When a group is removed from a shared or team folder, the folder is deleted from each group member's HCP Anywhere account. However, a member of the group is not removed from the folder if they were invited individually to the folder.

URI

/fss/public/share/removeGroup

HTTP Method

POST

Parameters

The share/removeGroup endpoint requires only request body parameters. The request body needs to be in JSON format.

Request body parameters:

•path (required) : String

The path to the shared or team folder.

•uniqueId (required): String

The unique identifying string assigned to the group by the group's authentication provider (AD or SAML). You can acquire a group's uniqueId using a using a share/invite/search request.

•providerId (required): String

The unique identifying string assigned to the group's authentication provider (AD or SAML). You can acquire a group's uniqueId using a using a share/invite/search request.

Example Request

This section contains an example of a share/removeGroup request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/removeGroup -H "Accept: application/json" -H "Content-Type: application/json" -H "Authorization: Bearer abc=" -H "X-HCPAW-FSS-API-VERSION: 3.0.0" -d '{"path": "/Group Project", "uniqueId":"S-1-5-21-3051028520-174195653-2777353501-2102","providerId":"2305843009213693953"}'

Example Response

For a successful share/removeGroup request, no information is returned in the response body.
