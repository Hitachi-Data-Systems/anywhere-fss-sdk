Delete team folder

This request deletes a team folder. When you delete a team folder using this endpoint, the folder is deleted from:

•Your HCP Anywhere account

•The HCP Anywhere accounts of each folder member

You can only delete a team folder with this endpoint if you are a Manager in the folder.

Note: If you want to delete a team folder from your HCP Anywhere account but not the HCP Anywhere accounts of the other folder members, use a path/delete request.

URI

/fss/public/share/team/delete

HTTP Method

POST

Parameters

The share/team/delete endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•path (required) : String

The path to the team folder.

Example Request

This section contains an example of a share/team/delete request in cURL. In this request, the access token is "abc=". In an actual request, the access token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/team/delete -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 4.0.0" -d '{"path": "/Group Project"}'

Example Response

For a successful share/team/delete request, no information is returned in the response body.
