Reject invitation

This request rejects an invitation to join a shared folder or team folder.

URI

/fss/public/share/rejectInvitation

HTTP Method

POST

Parameters

The share/rejectInvitation endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•fsId (required) : Long

A unique identifying number assigned to the invitation by HCP Anywhere. You can acquire an invitations fsId of an invitation using a share/listInvitations request.

Example Request

curl -kiX POST https://anywhere.example.com/fss/public/share/rejectInvitation -H "Accept: application/json" -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.3" -d '{"fsId": 1729382256910270467}'

Example Response

For a successful share/rejectInvitation request, no information is returned in the response body.
