List invitations

This request tists the pending shared and team folder invitations that have been sent to a given user.

URI

/fss/public/share/listInvitations

HTTP Method

POST

Parameters

There are no parameters for a share/listInvitations request.

Example Request

This section contains an example of a share/listInvitations request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/share/listInvitations -H "Accept: application/json" -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.3"

Description of Response

This section describes the contents of a share/listInvitations response.

Response content:

•invitations (required) : Object

A list of the current shared and team folder invitations for the user:

oinviter (required) : String

The person who invited the user to join the folder.

ofsId (required) : Long

A unique number assigned to the shared or team folder by HCP Anywhere.

osize (required) : Integer

The size of the folder in bytes.

orole (required) : Enum

The folder role of the member (VIEWER, COLLABORATOR, or MANAGER). The role identifies the operations a user can perform in the folder.

olabel (required) : String

The label of the shared or team folder.
