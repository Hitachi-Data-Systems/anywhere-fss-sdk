List identity providers

This request lists the identity providers (Active Directory and SAML) configured on an HCP Anywhere system.

Note: An authentication token is not needed in a provider/list request.

URI

/fss/public/provider/list

HTTP Method

POST

Parameters

There are no parameters for a provider/list request.

Example Request

This section contains an example of a provider/list request in cURL.

curl -kiX POST https://anywhere.example.com/fss/public/provider/list -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.3"

Example Response

This section contains an example of a response to a successful provider/list request.

{
    "providers":[
        {
           "id":"2305843009213693953",
           "name":"example.domain.com",
           "type":"ACTIVE_DIRECTORY"
         }
    ]
}

Description of Response

This section describes the contents of a provider/list response.

Response content:

•providers (required) : Object

A list of the identity providers for the HCP Anywhere server specified in the request:

oid (required) : Long

A unique identifying string assigned to the identity provider by HCP Anywhere.

oname (required) : String

For Active Directory, this is the name of the Active Directory domain connected to the HCP Anywhere server. For SAML, this is the name of the SAML identity as it is configured on the HCP Anywhere server.

otype (required) : Enum

The type of authentication provider (ACTIVE_DIRECTORY or SAML).
