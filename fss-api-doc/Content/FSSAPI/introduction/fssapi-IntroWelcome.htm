Welcome

Welcome to the Hitachi Content Platform Anywhere (HCP Anywhere) File Sync and Share API Reference Guide. The File Sync and Share API is a set of HTTP endpoints that allow you to build and integrate applications with HCP Anywhere. Each File Sync and Share API endpoint can be used to perform a different operation in HCP Anywhere. These operations include uploading and updating files, creating folders and links, and retrieving metadata about files and folders.

Intended audience

The HCP Anywhere File Sync and Share API Reference Guide is for:

•HCP Anywhere administrators who want to integrate custom or third-party applications with HCP Anywhere for their HCP Anywhere users

•HCP Anywhere users who want to integrate custom or third-party applications with their own HCP Anywhere account
