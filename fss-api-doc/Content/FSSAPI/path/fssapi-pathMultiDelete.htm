Delete multiple items

This request deletes multiple items in HCP Anywhere.

URI

/fss/public/path/multiDelete

HTTP method

POST

Parameters

The path/multiDelete endpoint requires only request body parameters. The request body needs to be in JSON format.

Request body parameters:

•entryList (required) : Object

The list of items you want to delete. For each file or folder you want to delete, you need to enter these two parameters:

opath (required) : String

The path of the file or folder.

oetag (required) : String

The etag of the file or folder. The etag identifies the version of the file or folder and is part of the file or folder metadata.

•recursive (optional) : Boolean

This parameter only applies to deleting a folder. If true, all the files and folders within the given folder are deleted. If false, and there are files or folders within a folder being deleted, the folder will not be deleted.

Example request

This section contains an example of a path/multiDelete request in cURL. In an actual request, the access token and etags are much longer.

curl -kiX POST https://anywhere.example.com/fss/public/path/multiDelete -H "Accept: application/json" -H "Content-Type: application/json" -H "Authorization: Bearer abc=" -H "X-HCPAW-FSS-API-VERSION: 4.0.0" -d '{"entryList": [{"path": "/Sales-Presentation.docx", "etag": "ePM=="}, { "path": "/Meeting Notes", "etag": "Rtq=="}], "recursive": true}'

Example response

This section contains an example of a response to a successful path/multiDelete request. In an actual response, the etags are much longer.

{
     "successfulDeletes":  [
         {
            "path": "a"
         },
         {
            "path": "x"
         }
       ],
    "failedDeletes": []
}

Description of Response

This section describes the contents of a path/multiDelete response.

Response content:

•successfulDeletes (required): Object

A list of the files and folders that were successfully deleted.

•failedDeletes (required): Object

A list of the files and folders that could not be deleted.
