Delete link

This request deletes a link.

URI

/fss/public/link/delete

HTTP Method

POST

Parameters

The link/delete endpoint requires only request body parameters. The request body must be in JSON format.

Request body parameters:

•path (required) : String

The path to the file or folder where the link is pointing.

•url (required) : String

The actual link.

Example Request

This section contains an example of a link/delete request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

curl -kiX POST https://anywhere.example.com/fss/public/link/delete -H "Content-Type: application/json" -H "Authorization: Bearer abc=" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.0" -d '{"path": "/Sales_Overview.pptx", "url": "https://anywhere.example.com/u/4vme5_B5nuPw-Z7n/Sales_Overview.pptx?l"}'

Example Response

For a successful delete, no information is returned.
