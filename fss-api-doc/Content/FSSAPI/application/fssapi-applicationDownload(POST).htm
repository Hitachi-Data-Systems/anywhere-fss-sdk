Download applications (POST)

This request downloads one of these HCP Anywhere applications from the HCP Anywhere server:

•Mac desktop application

•Windows desktop application

•Windows Outlook Add-in

The HCP Anywhere mobile applications are not available for downland from the HCP Anywhere server. However, the mobile applications for iOS, Android, and Windows Phone are available for download from each of their device's app store.

Note: Your organization may not allow you to use some, or possibly any, of the HCP Anywhere applications. If you organization is preventing you from using a particular app, you can download and install the app on your device but not register it.

URI

/fss/public/client/get

HTTP Method

POST

Parameters

The client/get (POST) endpoint requires only request body parameters. The request body needs to be in JSON format.

Request body parameters:

•client (required): Enum

The application to download from the HCP Anywhere server:

oWINDOWS — The desktop application for Windows.

oMAC — The desktop application for Mac.

oOUTLOOK_ADD_IN — The Outlook Add-in.

Example Request

This section contains an example of a client/get (POST) request in cURL. In this request, the authentication token is "abc=". In an actual request, the authentication token is much longer.

The "-o" option is supplied in each request. The "-o" option allows you to specify a local path on your computer where the file is saved instead of having the file streamed to your terminal in bytes. For the Windows desktop application and the Windows Outlook Add-in, make sure to enter "exe" as the extension for the installation file in the "-o" option. For the Mac desktop application, make sure to enter "dmg" as the extension for the installation file in the -o" option. Including the incorrect extension in the "-o" option corrupts the file.

Do not include an "i" in the request (such as "kiX"). This corrupts the file.

curl -kX POST https://anywhere.example.com/fss/public/client/get -H "Authorization: Bearer abc=" -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HCPAW-FSS-API-VERSION: 3.0.3" -d '{"client": "WINDOWS"}' -o "C:\JSMITH\Desktop\AW.exe"

Example Response

In response to a successful client/get (POST) request, no information is returned in the response body. Instead, the installation file is downloaded to the location that you specified in the "-o" option of the request.
